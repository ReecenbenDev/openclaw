# `reecenbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
reecenbot pairing list whatsapp
reecenbot pairing approve whatsapp <code> --notify
```

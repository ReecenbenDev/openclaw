# `reecenbot tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/tui)

## Examples

```bash
reecenbot tui
reecenbot tui --url ws://127.0.0.1:18789 --token <token>
reecenbot tui --session main --deliver
```

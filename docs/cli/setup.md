# `reecenbot setup`

Initialize `~/.reecenbot/reecenbot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
reecenbot setup
reecenbot setup --workspace ~/.reecenbot/workspace
```

To run the wizard via setup:

```bash
reecenbot setup --wizard
```
